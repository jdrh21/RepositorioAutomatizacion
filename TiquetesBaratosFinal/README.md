# TIQUETESBARATOS.COM
Ejercicio final semillero de automatización

El software realiza la busqueda, seleccion, reserva y compra de un vuelo de medellín a bogotá
Está desarrollado siguiendo el patrón ScreenPlay y basado en serenity, cucumber y selenium

Para generar el reporte de serenity se deben seguir una serie de pasos externos que serán especificados a continuación:

1 - Abrir el símbolo del sismte (cmd) como administrador
2 - Ubicarse en la raíz del proyecto
3 - Ejecutar el comando "gradle clean test"
4 - El informe de serenity quedará almacenado en el archivo target/site/serenity/index.html
